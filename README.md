## 说明

使用的库与数据库

* 使用[Bootstrap](http://getbootstrap.com/)作为前端库
* 开发环境使用[sqlite3](https://www.sqlite.org/)作为数据库
* 运行环境使用[Postgresql](http://postgresapp.com/)作为数据库
* 使用[qiniu](http://www.qiniu.com/)作为图片存储平台
